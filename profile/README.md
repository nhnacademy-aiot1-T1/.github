## 🪧 프로젝트 소개
### 🖥️ 스마트홈 기기 모니터링 및 관리 시스템
> 스마트홈을 사용중인 공간에서 전체적으로 기기들을 모니터링하고 관리할 수 있는 웹서버를 제공하는 것을 목적으로 합니다
- 주요기능
  - 기기별 센서 데이터 모니터링
  - 환경에 기반한 조명 제어
  - AI기반 적정 온습도 추천
  - 비정상적인 상황에 대한 알림 기능
  - 데이터 요약 및 리뷰
