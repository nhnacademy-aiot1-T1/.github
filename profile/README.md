## 🪧 프로젝트 소개
### 🖥️ 스마트 팩토리 모터관리
> 공장에서 돌아가는 모터와 관련된 센서 데이터를 모니터링할 수 있는 웹페이지와 모터에서 수집한 센서데이터를 바탕으로 AI를 활용하여 모터 상태를 진단하고 위험 예측을 제공하는 서비스입니다
- 주요기능
  - 전체 공간에 대한 모터 상태 모니터링 기능
  - 모터 별 데이터 현황 모니터링 기능
  - AI를 활용한 모터 상태 관리
