## 🪧 프로젝트 소개
### 🖥️ 스마트홈 기기 모니터링 및 관리 시스템
> 스마트홈을 사용중인 공간에서 전체적으로 기기들을 모니터링하고 기기를 제어할 수 있는 웹서버를 제공하는 것을 목표로 합니다 
- 주요기능
  - 센서 별 데이터 모니터링 기능
  - 공간 별 데이터 모니터링 기능
  - 자동 모드를 통한 기기 제어
  - 수동 모드를 통한 기기 제어
  - 기간별 데이터 차트 제공 및 분석
